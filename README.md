# Ai-Admin

### 平台简介：
Ai-Admin是使用Angular开发的快速开发平台，我们注重极致的用户体验和更高效率的迭代开发，UI简洁大方，技术紧随前沿。

### 核心依赖

| 依赖                   | 版本         |
| ---------------------- |------------|
| angular            | 13.1.0      |
| ng-zorro-antd           | 13.4.0   |
